Social Media Counters Plugin for WordPress.

== Description ==

Arkam Lite is a responsive WordPress plugin that displays stylish and elegant social media buttons with counters from your favorite social media networks using the included widget or shortcode anywhere in your website.

Full documentation is available [here](https://themient.com/support-s/arkam-lite/).

= Translations =
